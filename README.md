# Java / maven / AngularJs seed project

## Introduction

This project aims to be a good starting point for a java / maven / angularjs project.

check it out:

```bash
git clone https://github.com/IvoNet/java-angularjs-seed.git
```

## Installation

```bash
npm install
mvn package
```
